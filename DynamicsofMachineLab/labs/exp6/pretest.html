Balancing of multiple mass in single plane

Pre Test

Q1. The single plane balancing is applicable only to thin disks or rotors where unbalance is only
A   Two planes
B   One plane
C   Three planes
D   Four planes
Ans is B


Q2. Balancing of rotating and reciprocating parts of an engine is necessary when it runs at
A   slow speed
B   high speed
C   moderate speed
D   All the above
Ans is B


Q3. Balancing is a method of correcting or eliminating unwanted inertia forces and couples in rotating and reciprocating parts of the machine
A   True
B   False
Ans is A


Q4. A system of rotating masses is said to be in dynamic balance if any resultant centrifugal force or couple does not exist.
A   False
B   True
Ans is B


Q5. A rigid body is in equilibrium if
A   τnet = 0
B   Fnet = 0
C   Both a and b
Ans is C


Q6. Which of the following factors are not responsible for unbalancing in rotating systems?
A   Errors
B   Tolerance
C   Shape of the rotor
D   None of the above
Ans is D
