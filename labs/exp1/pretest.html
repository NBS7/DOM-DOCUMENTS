Proell Governor

Pre Test

Q1. Which among these is a pendulum type governor
A   Inertia Governor
B   Watt Governor
C   Porter Governor
D   Proell Governor

Ans is B


Q2. Which of the following Governor is not suitable for High speeds
A   Watt
B   Hartnell
C   Wilson Hartnell
D   Hartong

Ans is A


Q3. Which of the following is more appropriate if we remove the governor?
A   The engine wouldn't work at all
B   We Can't control the speed limit
C   It will take more fuel for the same distance
D   None of these

Ans is B


Q4. Which one of the following is a Dead weight type governor?
A   Porter Governor
B   Hartnell Governor
C   Wilson-Hartnell Governor
D   Hartung Governor

Ans is A


Q5. Which of the following statement is TRUE
A   The same equilibrium speed for m, M and H ball of smaller masses can be used in Proell governor compare to Porter governor.
B   The same equilibrium speed for m, M and H ball of larger masses can be used in Proell governor compare to Porter governor.
C   The same equilibrium speed for m, M and H ball of same masses can be used in Proell governor compare to Porter governor.
D   None of the above

Ans is A
