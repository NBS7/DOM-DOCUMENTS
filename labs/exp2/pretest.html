Porter Governor

Pre Test

Q1.Which one of the following is a Dead weight type governor?
A   Porter Governor
B   Hartnell Governor
C   Wilson-Hartnell Governor
D   Hartung Governor
Ans is A


Q2. The ratio of the height of porter governor (when the length of arms and links are equal) to the height of watt governor is (Where m is the mass of the ball and M is the mass of sleeve)
A   (m+M)/m
B   M/(m+M)
C   m/(m+M)
D   None of the above
Ans is A


Q3. Which type of governor is used in a petrol engine?
A   Pendulum type
B   Spring load type
C   Inertia type
D   Any of the above
E   SI engine doesn't have a governor
Ans is E


Q4. Governor is attached to the camshaft through
A   Bevel gear
B   Spur gear
C   Rack and pinion
D   Herringbone gear
Ans is A


Q5. Which of the following is more appropriate if we remove the governor:
A   The engine wouldn't work at all
B   We Can't control the speed limit
C   It will take more fuel for the same distance
D   None of these
Ans is B


Q6. The weight of a porter governor is
A   proportional to angular speed (ω)
B   inversely proportional to ω
C   proportional to ω2
D   inversely proportional to ω2
E   inversely proportional to ω2
Ans is D


Q7. The power of porter governor is
A   proportional to percentage increase speed (s)
B   inversely proportional to s
C   proportional to s2
D   inversely proportional to s2
E   independent of s
Ans is D


Q8. The effort of a porter governor is
A   proportional to percentage increase speed s
B   inversely proportional to s
C   proportional to s2
D   inversely proportional to s2
E   independent of s
Ans is A


Q9. For the same lift of sleeve, range of speed of proell governor as compared to porter governor is
A   Less
B   More
C   Equal
D   Half
E   Double
Ans is B
