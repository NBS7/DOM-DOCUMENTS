Dynamic analysis of slider crank mechanism

Pre Test

Q1. The number of links in a planer mechanism with revolute joints having 10 instantaneous centres is
A   2
B   4
C   3
D   5
Ans is C


Q2. The crank and lever mechanism will produce
A   Oscillating motion
B   Translating motion
C   Zig-zag motion
D   Rotary motion
Ans is A


Q3. If crank is fixed in single slider crank chain, we get
A   Beam engine
B   Oscillating engine
C   Rotary engine
D   Reciprocating engine
Ans is C


Q4. The oscillating cylinder engine can be obtained by fixing
A   Connecting rod
B   Lever
C   Slider
D   Crank
Ans is A


Q5. Klein's construction is useful to determine
A   crank has non-uniform angular velocity
B   crank has uniform angular velocity and angular acceleration
C   crank has a uniform angular velocity
D   there is no such criterion
Ans is C
